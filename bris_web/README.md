# BrisWeb

![CI Status](https://github.com/jhonataningesis/BRISMAR_APP/actions/workflows/web_admin_ci.yml/badge.svg)
[![codecov](https://codecov.io/gh/jhonataningesis/BRISMAR_APP/branch/main/graph/badge.svg)](https://codecov.io/gh/jhonataningesis/BRISMAR_APP)

Panel de administración web para el control de zarpes, compras, gastos y cuadres marítimos.

## Características
- Gestión de transitos y cuadres.
- Paginación de recursos.
- Pruebas unitarias configuradas.
- CI/CD usando GitHub Actions con comprobaciones de seguridad.
