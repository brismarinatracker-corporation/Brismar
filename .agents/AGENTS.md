# Reglas del Proyecto - Brismar App

## Ramas y Responsabilidades
- **Rama `develop-belen`**: Reservada exclusivamente para tareas y cambios de diseño visual y de interfaz de usuario (UI/Frontend). No se debe implementar, modificar ni alterar lógica de negocio aquí.
- **Rama `develop`**: Contiene la lógica del negocio oficial y la arquitectura del backend/servicios (cuadres, modelos, repositorios, etc.). Todos los desarrollos de funcionalidad pura se importan de esta rama.
