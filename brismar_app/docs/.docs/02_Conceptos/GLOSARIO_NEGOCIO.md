## Términos de la bahía

| Término | Qué significa |
|---|---|
| **Bahía** | Zona del puerto donde se descarga la pesca |
| **Embarcación** | Barco pesquero |
| **Producto** | Tipo de pescado (Pota, Merluza, Jurel...) |
| **Kilos** | Cantidad de pesca en kg |
| **Precio/kg** | Valor en soles por kilo |
| **Muelle** | Punto donde se descarga |
| **Placa de carro** | Matrícula del camión refrigerado |
| **Pesador** | Persona que pesa la pesca |

---

## Gastos operativos

| Gasto | Qué es |
|---|---|
| **Hielo** | Para conservar la pesca |
| **Personal** | Cargadores y pesadores |
| **Flete** | Transporte (camión) |
| **Agua** | Lavar el producto |
| **Clorox** | Desinfección |
| **Facturación** | Costos administrativos |
| **Apoyo** | Personal extra |
| **Otros** | Gastos varios |
