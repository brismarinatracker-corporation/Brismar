# Brismar App — Cliente Móvil (Flutter)

Este directorio contiene todo el código fuente de la aplicación móvil para operadores de bahía.

## Arquitectura
* **Framework:** Flutter
* **Patrón de Diseño:** Clean Architecture + Repository Pattern
* **Gestión de Estado:** Riverpod
* **Persistencia:** Supabase (Online) + SQLite/SecureStorage (Offline-First)

## Estructura
Todo el código principal se encuentra en la carpeta `lib/`, dividida en `nucleo/` y `modulos/`.

*Nota: Para conocer las reglas estrictas de desarrollo de la empresa, revisa los documentos en la carpeta `.docs/` ubicada en la raíz del repositorio general.*
