# ⚓ BRISMAR APP

> Sistema de gestión pesquera para **Negocios Brismar S.R.L.**

---

## ¿Qué es?
Una app móvil que digitaliza el registro de pesca en bahía:
- Registrar kilos, precios y embarcaciones
- Controlar gastos operativos
- Calcular la utilidad neta automáticamente
- Generar reportes PDF
